# movies-service

Сервис для поиска и просмотра фильмов с фильтрацией по рейтингу, году и жанрам с использованием готового [API](https://poiskkino.dev/). 
Поддерживает добавление в избранное и сравнение фильмов между собой. 

## Инструкция

### Сборка проекта
```
npm run build
```

### Запуск
```
npm run dev
```
После, сервис будет доступен по адресу 

```
http://localhost:5173
```
